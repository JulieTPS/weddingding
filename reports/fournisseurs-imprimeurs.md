## 4. Chine (Alibaba) — Prix le plus bas, contraintes logistiques

| Fournisseur | Prix 100 cartes | Finitions | MOQ | Délai | Usage recommandé |
|---|---|---|---|---|---|
| **Guangzhou Fashion Cards** | ~€450-600 TTC livraison (finitions luxe) | Dorure à chaud, laser cut, gaufrage, soie, vellum | 100 pcs | 15-25j | Prototypes, tests finitions |
| **Guangzhou Ruihang** | ~€300-500 TTC | Dorure, pelliculage, 350g | 100 pcs | 15-25j | Prototypes |
| **AliExpress (générique)** | ~€1.50-3/carte | Basique à moyen | 50-200 pcs | 20-35j | Non recommandé pour clients |

**Prix unitaire Alibaba avec dorure à chaud** : ~$4.80-5.20/pièce pour 100 ex (vs €8-15/pièce en France pour même finition).

**Pourquoi ne pas utiliser pour les commandes clients :**
- Délai 15-25 jours incompatible avec attentes clients (veulent 1-2 semaines)
- Chaque commande WeddingDing est unique (noms/dates différents) → les fournisseurs chinois préfèrent les séries identiques
- Contrôle qualité difficile à distance
- Risque de non-conformité (couleurs, grammage)

**Usage optimal :** commander 50-100 cartes de prototype pour tester les finitions premium (dorure laser cut, vellum, soie) avant de les intégrer à l'offre Prestige.

## 6. Print-on-Demand API (UK/Europe) — Pour automatisation future

| Fournisseur | Modèle | Prix unitaire | Délai EU | Particularité |
|---|---|---|---|---|
| **Prodigi** | API REST, white-label, 50+ labs EU | ~€2-4/carte (400g) | 3-5j | Zéro stock, branding retiré, intégration code |
| **Printify** | Marketplace de labs EU | ~€1.50-3/carte | 5-8j | Choix de labs, finitions limitées |
| **Printful** | Infrastructure propre | ~€2.50-4/carte | 4-7j | Qualité constante, peu de 350g+ |

**Prodigi est le plus intéressant à terme** : API REST permettant d'automatiser le flux complet — commande client → génération PDF → envoi auto à Prodigi → livraison directe au couple. Élimine toute manipulation manuelle. Mais finitions premium (dorure à chaud, vernis sélectif) non disponibles pour l'instant.
