---

## RÉPONSE À LA QUESTION CENTRALE

**En juin 2026, peut-on créer une vidéo TikTok convaincante d'une carte NFC mariage sans jamais filmer ?**

**Réponse honnête : Oui pour les VISUELS STATIQUES, Partiellement pour la VIDÉO courte, Non pour la scène NFC-tap précise.**

- Les **images fixes** (flat lay, carte sur marbre, main tenant l'invitation) atteignent la qualité premium avec Midjourney V7 ou Freepik Mystic.
- Les **vidéos courtes** (3-8 secondes, mouvement doux, atmosphérique) fonctionnent bien avec Kling 3.0.
- La **scène NFC-tap spécifique** (main + carte + téléphone + interaction de contact physique) est exactement le type de scène où l'IA vidéo échoue le plus fréquemment en 2026 — trop d'objets, trop d'interaction précise, physique de contact.

**La stratégie recommandée** : image-to-video (générer d'abord une image fixe premium, puis animer avec subtilité) surperforme largement le text-to-video pour cette scène.

## FEUILLE DE ROUTE — SCÈNE NFC TAP MARIAGE

### Étape 1 — Générer l'image fixe premium (Midjourney V7)

Prompt type :
```
elegant hand holding a premium cream wedding invitation card with gold foil embossing,
approaching a modern iPhone, marble background, soft natural lighting, editorial photography
style, f/1.8 bokeh, high-end lifestyle
```

Utiliser Freepik Mystic pour les détails de peau de main si Midjourney donne des doigts imparfaits.

### Étape 2 — Animer l'image fixe (Kling image-to-video)

Partir de l'image fixe générée → Kling image-to-video → prompt de mouvement minimal :
```
hand slowly brings card toward phone, subtle movement, camera fixed, 4 seconds
```
Générer 5-10 variantes et sélectionner la meilleure. L'image-to-video performe significativement mieux que text-to-video sur les interactions précises.

### Étape 3 — Site web sur l'écran du téléphone

- Option A : Screen recording de ton mockup de site → CapCut pour incrustre l'écran
- Option B : Midjourney génère une image du téléphone affichant le site

### Étape 4 — Assemblage TikTok (CapCut)

- B-roll : clip NFC-tap 4-8 secondes
- Hook texte animé : "Cette carte change tout..."
- Voix off ElevenLabs FR
- Sous-titres auto CapCut
- Musique tendance TikTok (bibliothèque intégrée)
- Total vidéo : 15-25 secondes

## LIMITES HONNETES DE L'IA VIDEO EN 2026 (POUR LE MARCHE MARIAGE)

Les 6 problèmes structurels qui subsistent :

1. **Cohérence d'objet** : plus il y a d'objets dans une scène, plus la consistance visuelle se dégrade
2. **Dégradation qualité dans le temps** : visible après quelques secondes de clip
3. **Représentation exacte du produit** : l'IA génère "quelque chose qui ressemble" à ta carte, pas ta carte exacte
4. **Physique de contact** : tenir un objet, appuyer sur une surface = erreurs fréquentes
5. **Biais des données d'entraînement** : outputs Western-centric
6. **Marché mariage = émotionnel et artisanal** : le contenu AI visible comme tel peut contre-signaler pour ce segment précis

**Stratégie de mitigation :** Le marché de test pré-lancement est exactement le bon moment pour du contenu IA. Si la validation fonctionne, le premier revenu permet de financer une vraie session photo/vidéo pour créer du contenu hero authentique.
